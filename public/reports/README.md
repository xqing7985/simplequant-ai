# 研报 PDF 文件目录

## 中文研报
将中文 PDF 研报文件放在 `zh/` 目录下

例如：
- `2026-03-citic-factor-investing.pdf`

## 英文研报
将英文 PDF 研报文件放在 `en/` 目录下

例如：
- `2026-03-citic-factor-investing.pdf`

## 命名规范

建议使用以下命名格式：

```
YYYY-MM-<issuer>-<topic>.pdf
```

其中：
- `YYYY-MM` - 研报发布日期
- `<issuer>` - 发布机构简称（如 citic、cicc 等）
- `<topic>` - 研报主题
